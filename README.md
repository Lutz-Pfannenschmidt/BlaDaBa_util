# KlaDaBa.user.js

This userscript is designed for the KlaDaBa/Exam database of the Technical Faculty at Albert Ludwig University of Freiburg. You can access the database at [db.fachschaft.tf](https://db.fachschaft.tf).

[**Install the script here**](https://github.com/Lutz-Pfannenschmidt/BlaDaBa_util/raw/refs/heads/main/KlaDaBa.user.js)

## Features
- Enables middle-click functionality on table entries to open them in a new tab.
